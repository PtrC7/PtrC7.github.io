Digital Clock Lab

Final Lab in my EE Lab 1 class (EE 221). Here I created a 24 hour digital clock on a breadboard using various integrated circuits and logic gates. Using binary coded decimals, I displayed the decimals on seven-segment displays. Using a function generator, a clock signal was applied to the board which displayed the time.

The video above shows my clock running from an arduino board which is acting as the function generator.
